План тестирования

Стратегия
Модульное тестирование каждого класса/функции.
Использование Google Test.
Тесты пишутся до реализации (TDD).

Набор тестов
Тесты для DateValidator
Корректные даты: 01.01.2000, 29.02.2020, 31.12.2023.
Некорректные: 32.01.2000, 00.01.2000, 29.02.2021, 31.04.2000, "abcd".

Тесты для InputReader
readString: проверка, что возвращается введённая строка.
readInt: корректный ввод 123, ошибка при "abc", затем исправление.
readDouble: аналогично.
 readDate: корректная дата, затем некорректные форматы.

Тесты для ConsoleMenu
Навигация стрелками (эмуляция через замену _getch() сложна, поэтому тестируется только логика выбора индекса без реального ввода).
Выбор Enter возвращает правильный индекс.
